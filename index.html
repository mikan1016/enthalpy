エンタルピー変化に関する問題

問題

  1. エンタルピーとは何ですか？エンタルピー変化が正のときと負のとき、どのような反応が起こるか説明してください。
  2. 次の反応におけるエンタルピー変化を計算してください：
    H2(g) + 1/2 O2(g) → H2O(l)
    反応における標準生成エンタルピーは以下の通りです：
    • H2O(l): -285.8 kJ/mol
    • H2(g): 0 kJ/mol
    • O2(g): 0 kJ/mol
  3. 次の反応式に基づいてエンタルピー変化を求め、反応が吸熱反応か発熱反応か判断してください：
    CH4(g) + 2 O2(g) → CO2(g) + 2 H2O(l)
    各物質の標準生成エンタルピーは以下の通りです：
    • CH4(g): -74.8 kJ/mol
    • CO2(g): -393.5 kJ/mol
    • H2O(l): -285.8 kJ/mol
    • O2(g): 0 kJ/mol
  4. 次の反応式における生成エンタルピーを求めてください：
    C(s) + O2(g) → CO2(g)
    標準生成エンタルピーは以下の通りです：
    • C(s): 0 kJ/mol
    • CO2(g): -393.5 kJ/mol
    • O2(g): 0 kJ/mol
  5. 次の反応に基づいてエンタルピー変化を計算し、この反応がどのようなエネルギーの変化を伴うか述べてください：
    2 H2O(l) → 2 H2(g) + O2(g)
    各物質の標準生成エンタルピーは以下の通りです：
    • H2O(l): -285.8 kJ/mol
    • H2(g): 0 kJ/mol
    • O2(g): 0 kJ/mol
  6. 次の反応の標準生成エンタルピーが求められているとき、反応のエンタルピー変化を求めてください：
    N2(g) + 3 H2(g) → 2 NH3(g)
    各物質の標準生成エンタルピーは以下の通りです：
    • N2(g): 0 kJ/mol
    • H2(g): 0 kJ/mol
    • NH3(g): -45.9 kJ/mol
