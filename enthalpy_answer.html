エンタルピー変化に関する解答

解答

  1. **エンタルピー**は、物質が持つエネルギーの総量を示す指標です。エンタルピー変化が正のとき（ΔH > 0）、反応は吸熱反応であり、外部からエネルギーを吸収します。エンタルピー変化が負のとき（ΔH < 0）、反応は発熱反応であり、エネルギーを外部に放出します。
  2. H2(g) + 1/2 O2(g) → H2O(l) のエンタルピー変化:
    ΔH = (生成物のエンタルピー) - (反応物のエンタルピー)
    ΔH = [-285.8 kJ/mol] - [0 kJ/mol + 0 kJ/mol]
    ΔH = -285.8 kJ/mol（発熱反応）
  3. CH4(g) + 2 O2(g) → CO2(g) + 2 H2O(l) のエンタルピー変化:
    ΔH = [(CO2(g) + 2 H2O(l))] - [(CH4(g) + 2 O2(g))]
    ΔH = [(-393.5) + 2(-285.8)] - [(-74.8) + 0]
    ΔH = -963.1 kJ/mol（発熱反応）
  4. C(s) + O2(g) → CO2(g) のエンタルピー変化:
    ΔH = (生成物のエンタルピー) - (反応物のエンタルピー)
    ΔH = [-393.5 kJ/mol] - [0 kJ/mol + 0 kJ/mol]
    ΔH = -393.5 kJ/mol（発熱反応）
  5. 2 H2O(l) → 2 H2(g) + O2(g) のエンタルピー変化:
    ΔH = (生成物のエンタルピー) - (反応物のエンタルピー)
    ΔH = [0 + 0] - [2(-285.8 kJ/mol)]
    ΔH = +571.6 kJ/mol（吸熱反応）
  6. N2(g) + 3 H2(g) → 2 NH3(g) のエンタルピー変化:
    ΔH = (生成物のエンタルピー) - (反応物のエンタルピー)
    ΔH = [2(-45.9 kJ/mol)] - [0 kJ/mol + 0 kJ/mol]
    ΔH = -91.8 kJ/mol（発熱反応）
